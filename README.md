# Brincolines-Markitos

## Sistema de Gestión de Renta de Mobiliario

## Descripción
Este proyecto consiste en el diseño e implementación de una base de datos relacional para gestionar el inventario y los procesos de alquiler de un negocio familiar dedicado a la renta de eventos. El sistema permitirá administrar el catálogo de productos (sillas, mesas, carpas, brincolines, etc.), registrar clientes y controlar las reservaciones por fecha para asegurar la disponibilidad del equipo.

## Motivación
Actualmente, el control de las rentas se realiza de forma informal, lo que ocasionalmente genera problemas como:
- Desconocimiento del equipo real disponible para una fecha futura.
- Dificultad para rastrear qué cliente tiene qué mobiliario.
- Errores que se pueden cometer en la agenda (dobles reservaciones).
- Administrar cambios que proponga el cliente (mas equipo o cancelar un evento)

El objetivo es digitalizar este proceso para optimizar la logística, evitar conflictos de agenda y tener un histórico claro de las rentas y ganancias.

## Tech Stack
- **Motor de Base de Datos:** MySQL
- **Diseño:** Diagramas Entidad-Relación (E-R).
- **Lenguaje de Consultas:** SQL.
