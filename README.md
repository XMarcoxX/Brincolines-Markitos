# Brincolines-Markitos

## Sistema de Gestión de Renta de Mobiliario

## Descripción
Este proyecto trata en el diseño e implementación de una base de datos para revisar el inventario del negocio familiar dedicado a la renta de eventos. Se administrara los productos (sillas, mesas, carpas, brincolines, etc.), registrar clientes nuevos y asegurar la disponibilidad del mobiliario.

## Motivación
Aveces se pierde el control de las rentas, lo que genera problemas como:
- Dificultad para saber qué cliente tiene qué mobiliario.
- Errores que se pueden haber en la agenda.
- Administrar cambios que quiera el cliente como mas mobiliario o quitar mobiliario

## Tech Stack
- **Base de Datos:** MySQL
- **Diseño:** Diagramas Entidad-Relación (E-R).
